# dsh-card-details

Web component de tarjeta de detalles del DSH Design System. Incluye línea de color superior, header con ícono y título, acordeón expandible, y área de footer con acciones.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `color` | String | `"neutral"` | Color: neutral, blue, purple, slate, cav, ffmm, apv |
| `open` | Boolean | `true` | Si el acordeón interno está expandido |
| `accordeonable` | Boolean | `false` | Activa el botón colapsar/expandir (solo en móvil) |
| `icon` | String | — | Nombre del ícono FontAwesome (ej: fa-user) |
| `footer-align` | String | `"fluid right"` | Alineación del footer: left, right, center, fluid |

## Slots

| Slot | Descripción |
|------|-------------|
| `title` | Título del card (strong, h4, p) |
| `content` | Contenido principal |
| `footer` | Botones de acción |
