# dsh-message

Web component de mensaje del DSH Design System. Combina tres variantes de mensaje en un solo componente controlado por el prop `variant`.

## Variantes disponibles

| Variant | Descripción |
|---------|-------------|
| `info` | Mensaje informativo simple con ícono azul y fondo blanco |
| `informative` | Fondo azul claro, ícono info |
| `success` | Fondo verde claro, ícono check |
| `error` | Fondo rojo claro, ícono X |
| `warning` | Fondo naranja claro, ícono triángulo |
| `important` | Borde punteado, fondo índigo claro, sin ícono automático |

## Slots

| Slot | Descripción |
|------|-------------|
| `title` | Título del mensaje |
| `content` | Contenido principal |
| `footer` | Acciones o enlaces al pie |
| `icon` | Ícono personalizado (solo para variant `important`) |
