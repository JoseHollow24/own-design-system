# dsh-menu-vertical

Web component de menú lateral vertical del DSH Design System. Soporta ítems con sub-ítems y modo responsive (mobile/tablet/desktop).

## Default

<Canvas of={MenuVerticalStories.Default} />

## Con item activo

<Canvas of={MenuVerticalStories.WithActiveItem} />

## Componentes

| Componente | Descripción |
|---|---|
| `dsh-menu-vertical` | Contenedor del menú con header responsive y slot `menu-item` |
| `dsh-menu-vertical-item` | Ítem de menú principal con soporte para sub-ítems |
| `dsh-menu-vertical-subitem` | Sub-ítem del menú |

## Props de dsh-menu-vertical

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `open-mobile` | Boolean | `false` | Abre el menú en modo mobile |

## Props de dsh-menu-vertical-item

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `item-value` | String | `""` | Texto del ítem |
| `item-identifier` | String | `""` | Identificador único del ítem |
| `href` | String | `"#"` | URL de destino |
| `target` | String | `"_self"` | Target del enlace |
| `active` | Boolean | `false` | Estado activo |
| `desktop-icon-name` | String | `"bullet"` | Icono Font Awesome para desktop (ej: `fa-house`) o `"bullet"` |
| `mobile-icon-name` | String | `"bullet"` | Icono Font Awesome para mobile |
| `is-home-item` | Boolean | `false` | Marca el ítem como home (se oculta en mobile/tablet) |
| `height-menu` | String | `""` | Altura del menú (asignada automáticamente) |

## Props de dsh-menu-vertical-subitem

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `href` | String | `"#"` | URL de destino |
| `target` | String | `"_self"` | Target del enlace |
| `active` | Boolean | `false` | Estado activo |
| `sub-item-identifier` | String | `""` | Identificador único del sub-ítem |

## Slots de dsh-menu-vertical

| Slot | Descripción |
|------|-------------|
| `menu-item` | Ítems `dsh-menu-vertical-item` |
| `menu-avatar` | Avatar/logo para el header desktop |
| `button-header-mobile` | Botón adicional en el header mobile |

## Slots de dsh-menu-vertical-item

| Slot | Descripción |
|------|-------------|
| `sub-item` | Sub-ítems `dsh-menu-vertical-subitem` |
