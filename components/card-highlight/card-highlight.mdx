# dsh-card-highlights

Web component de tarjetas destacadas del DSH Design System. Presenta una serie de elementos visuales apilados con efecto de profundidad, donde el primero se muestra en primer plano.

## Uso

```html
<dsh-card-highlights>
  <dsh-card-highlights-item slot="tab" color="a6">
    <svg slot="icon">...</svg>
    <span slot="label">Primer elemento</span>
  </dsh-card-highlights-item>
  <dsh-card-highlights-item slot="tab" color="r6">
    <svg slot="icon">...</svg>
    <span slot="label">Segundo elemento</span>
  </dsh-card-highlights-item>
</dsh-card-highlights>
```
