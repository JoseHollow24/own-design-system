Web component de botón del DSH Design System. Construido con Atomico sobre Shadow DOM.

Soporta variantes **primary**, **secondary** y **tertiary**, colores **blue** y **yellow**, y estados como disabled, loading, full y fluid.

## React — Cómo usarlo

El componente `<dsh-button>` es un Web Component estándar, consumible desde cualquier framework.
El evento que dispara se llama `buttonClick` (CustomEvent, `bubbles: true`, `composed: true`).

<Canvas of={ButtonStories.ReactUsage} />

El tab **Show code** muestra los dos patrones de consumo:
- **React 19** — prop `onbuttonclick` directamente en JSX (soporte nativo de custom elements)
- **React 18** — `useRef` + `addEventListener('buttonClick', handler)`
